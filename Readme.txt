Чтобы создать ехе файл необходимо:
1. Открыть файл SaveImage.sln при помощи Visual Studio.
2. Конфигурация решения - Release.
3. Пуск(нажать F5 в Visual Studio).
4. Перейти в папку SaveImage/bin.
5. Папка Release - готовое приложение со всеми необходимыми библиотеками и ехе файлом
Чтобы создать БД MSSQL для этой программы надо:
1. Создать БД по имени db
2. Создать пользователя Skyline, пароль 123456789
3. Использовать скрипт script.sql для создания таблиц
Чтобы изменить имя сервера или базы данных, необходимо открыть SaveImage.exe.config в папке SaveImage/bin/Release и изменить параметр connectionString
